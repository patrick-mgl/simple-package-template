# my_bank

Description. 
The package my_bank is used to:
	- realize cash removal operations
	- realize cash deposit operations

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install my_bank

```bash
pip install my_bank
```

## Usage

```python
from my_bank import operations
operations.deposit()
operations.removal()
```
